# 💈 Barbearia Elite

Um site moderno e responsivo desenvolvido para uma barbearia fictícia, com foco em **design elegante, experiência do usuário e conversão de clientes**.

Este projeto foi criado como parte do meu **portfólio de desenvolvimento web**, simulando um site real que poderia ser utilizado por uma barbearia profissional.

---

## ✨ Demonstração do Projeto

O site apresenta uma estrutura completa com:

* Hero section moderna
* Sessão sobre a barbearia
* Cards de serviços
* Galeria interativa
* Botão de contato direto no WhatsApp
* Layout responsivo para celular

---

## 🚀 Funcionalidades

✔ Menu responsivo para dispositivos móveis
✔ Animações ao rolar a página
✔ Galeria com visualização de imagem ampliada (Lightbox)
✔ Botão flutuante de WhatsApp
✔ Design moderno com foco em conversão de clientes
✔ Layout totalmente responsivo

---

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* AOS (Animate On Scroll)
* Font Awesome
* Google Fonts

---

## 📱 Responsividade

O site foi desenvolvido com foco em **experiência mobile-first**, garantindo boa navegação em:

* 📱 Smartphones
* 📲 Tablets
* 💻 Computadores
